			NodeJS
                =======================
- NodeJS is used to create server side application by using JavaScript or reast Api.
- NodeJs is JavaScript run time enviorment (Not a framework or Library ).
-Runs on Chrome's V8 engine.

What you should know before starting the NodeJS?
- Javascript (fxn, callbacks, promises .....).
- NodeJS is written in c++

Why should we use NodeJS?
- it is light-weight and open source
- sync(synchronus) --> code execute line by line it is called synchronus.
- async(asynchronus) --> code execute without any restriction it is called asynchronus

- same language we can use both side client side and server side.
